# Indi-CLIM
India Climate Live-Intelligence Model

An advanced analytical framework designed for the **ISRO Hackathon**, focusing on live-intelligence climate modeling, environmental tracking, and predictive visualization across the Indian subcontinent.

## 🛰️ Project Overview
Indi-CLIM leverages satellite data paradigms, geospatial intelligence, and predictive modeling to provide real-time climate insights. The system processes complex environmental variables—such as temperature anomalies, precipitation cycles, and atmospheric patterns—to assist researchers, disaster management teams, and policymakers.

### Key Features
* **Live-Intelligence Data Pipelines**: Engineered to ingest and simulate real-time climate metrics.
* **Predictive Modeling**: Employs Python-based analytical models to forecast regional climate variations.
* **Interactive Visualization**: Front-end interface built with JavaScript, HTML, and CSS for mapping climate risks.
* **Mock Data Synthesis**: Integrated testing suite to simulate satellite telemetry and weather sensor nodes.

## 🚀 Getting Started

### Prerequisites
* Python 3.8 or higher
* A modern web browser with JavaScript enabled

### Installation & Local Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com
   cd Indi-CLIM
   ```

2. **Set up a virtual environment:**
   ```bash
   python -m venv .venv
   source .venv/bin/activate  # On Windows use: .venv\Scripts\activate
   ```

3. **Generate simulated climate dataset:**
   ```bash
   python mock_data_generator.py
   ```

4. **Launch the frontend dashboard:**
   Open the core HTML interface file located inside the `indi-clim/` directory directly in your web browser.
